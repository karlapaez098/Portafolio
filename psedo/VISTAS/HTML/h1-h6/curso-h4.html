Curso HTML - <h4>

La etiqueta <h4> es muy poco común para la creación de páginas web en la actualidad. Con los diferentes lenguajes que existen en la actualidad para la creación de páginas web (PHP,JS,PHP) no se utiliza <h4>. Aún asi se la utiliza como subsección del <h3>.

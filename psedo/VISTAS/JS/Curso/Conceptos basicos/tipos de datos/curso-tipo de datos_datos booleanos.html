Curso JS - Tipos de datos

Datos Booleanos

En JavaScript booleano, puede tener uno de dos valores, verdadero o falso . Éstos son útiles cuando necesita un tipo de datos que solo puede tener uno de dos valores, como Sí / No, Activado / Desactivado, Verdadero / Falso.

Ejemplo:

El valor booleano de 0 (cero), nulo , indefinido , cadena vacía es falso . Todo lo que tenga un valor "real" es cierto.

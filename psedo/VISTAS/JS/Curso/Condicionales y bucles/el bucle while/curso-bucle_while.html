Curso JS - El bucle while

El bucle while

El while repite el bucle a través de un bloque de código, siempre que una condición especificada es verdadera.

Sintaxis:

La condición puede ser cualquier declaración condicional que devuelva verdadero o falso.

Considere el siguiente ejemplo.

El bucle continuará ejecutándose mientras i sea menor o igual a 10. Cada vez que se ejecute el bucle, aumentará en 1.
Esto generará los valores de 0 a 10.

Tenga cuidado con las condiciones de escritura. Si una condición es siempre verdadera, el ciclo se ejecutará para siempre.

Si olvida aumentar la variable utilizada en la condición, el ciclo nunca terminará.

Asegúrese de que la condición en un ciclo while eventualmente se vuelva falsa.
